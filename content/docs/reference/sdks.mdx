### Node.js SDK

The Node.js TypeScript SDK for Authula provides a seamless way to interact with the Authula server.

With support for client-side plugins, it gives you powerful tools to enhance your authentication workflows and interact with your Authula server in a flexible and extensible manner.

You can view the SDK here: [Node.js SDK](https://www.npmjs.com/package/authula)
